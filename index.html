• Create a repository: git init [path/to/project]
  • Clone from a repository: git clone [url/from/repository] [path/to/project]
  • Add Changes or new Files to staging Area: git add test.txt
  • Add all to staging Area: git add .
  • Commit staged Changes: git commit -m "test msg"
  • See status of git -> see how many commits and changes: git status
